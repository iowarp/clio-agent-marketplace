title: Recommend Skip Flag Or User Question
description: Decide whether risky fields should be skipped, flagged, or escalated.
---

Use this skill when conversion policy finds unsafe fields. Recommend skipping
fields that cannot be represented safely, flagging lossy but acceptable
conversions, or asking the user when scientific meaning depends on the choice.

Keep the recommendation tied to field names and risks.
