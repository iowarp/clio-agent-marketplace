title: Attribute I/O Regression
description: Attribute an HPC I/O regression from aligned metric evidence.
---

Build the root-cause hypothesis from aligned evidence. Prefer explanations that
account for the largest changed timings while remaining consistent with stable
workload metrics.

State the confidence and caveats. If the evidence only supports a likely cause,
say so. Mention follow-up measurements that would confirm the hypothesis.
