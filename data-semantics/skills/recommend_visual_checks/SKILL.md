title: Recommend Visual Checks
description: Choose visual checks that match the inspected data.
---

Use this skill when a dataset needs visual review. Match the plot type to the
data: histograms for distributions, scatter plots for paired numeric fields,
line plots for time or ordered dimensions, maps for spatial coordinates, and
heatmaps for gridded arrays.

Recommend only visualizations that can be backed by available variables.
