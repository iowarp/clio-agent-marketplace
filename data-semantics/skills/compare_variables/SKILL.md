name: compare_variables
title: Compare Variables
description: Compare scientific variables with units, ranges, and quality caveats.
---

Use this skill when comparing columns, datasets, arrays, or variables across
scientific files. Preserve each variable name exactly and report units, shape,
range, missingness, and suspicious outliers when available.

Separate comparable evidence from incompatible evidence. If variables differ in
units, resolution, time range, or coordinate system, state that before drawing
conclusions.
