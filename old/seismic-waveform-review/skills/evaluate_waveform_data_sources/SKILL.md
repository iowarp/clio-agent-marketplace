name: evaluate_waveform_data_sources
title: Evaluate Waveform Data Sources
description: Evaluate seismic data sources and preserve provenance.
---

Use this skill during waveform discovery. Compare NDP catalog records, direct
waveform resources, EarthScope or other public sources, delivery type, size,
time window, station/channel, and provenance.

If a source fails, return structured failure context to the parent.
