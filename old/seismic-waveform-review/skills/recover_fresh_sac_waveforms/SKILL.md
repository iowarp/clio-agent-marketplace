title: Recover Fresh SAC Waveforms
description: Recover waveform data from an alternate public source when catalog staging fails.

Use this skill when an NDP catalog record fails to deliver a usable waveform.
Try a bounded public waveform source or direct download path while preserving
the failed source, alternate source, time window, station/channel, and caveats.

Return control to the parent with both failure and recovery provenance.
