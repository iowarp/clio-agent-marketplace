name: inspect_ogc_collections
title: Inspect OGC Collections
description: Inspect OGC API collection metadata for remote-sensing discovery.
---

Use this skill when reviewing an OGC API, STAC-like, or catalog collection.
Record collection identifiers, titles, spatial extent, temporal extent, asset
types, query limits, and missing metadata.

Keep catalog evidence separate from downloaded-data evidence. Do not claim data
quality from metadata alone.
